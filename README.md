# Cartinhas RPG – Plataforma de Cartas Digitais Estratégicas

## Visão Geral

Cartinhas RPG é uma plataforma interativa de cartas digitais com temática de RPG, que combina **colecionismo, estratégia e evolução diária**. Cada carta representa um personagem único, ambientado em cenários distintos e com raridades visuais que refletem seu poder.

O objetivo do jogador é coletar cartas diariamente, escolher quais reforjar ou fundir, e criar cartas cada vez mais poderosas por meio de decisões estratégicas.

---

## Público-Alvo

* Pessoas que gostam de mecânicas de evolução de personagens.
* Usuários que buscam uma experiência diária de estratégia simples e envolvente.

---

## Funcionalidades

* **Cartas Diárias**: Receba 5 cartas únicas por dia com personagens (Mago, Arqueiro, Guerreiro), cenários (Planície, Floresta, Montanha) e raridade visual (Comum, Bom, Ótimo, Top, Perfeito).
* **Reforjar Cartas**: Combine duas cartas semelhantes para obter a carta com os melhores atributos de ambas.
* **Fundir Cartas**: Combine duas cartas diferentes para gerar uma carta de raridade superior e atributos otimizados.
* **Atributos Inspirados em RPG**: Cada carta possui atributos como Constituição, Força, Destreza, Inteligência, Sabedoria e Carisma.
* **Visualização Interativa**: Atras de cada carta, um hexágono mostra os atributos de forma gráfica, facilitando a comparação rápida.
* **Ações Diárias Limitadas**: Cada ação (reforjar, fundir, cartas diárias) pode ser realizada uma vez por dia, incentivando decisões estratégicas.

---

## Como Jogar

1. **Login/Cadastro**: Crie sua conta ou entre com seu usuário existente.
2. **Receba Cartas Diárias**: Acesse sua seleção diária de 5 cartas.
3. **Escolha Estratégica**:

   * Reforjar cartas similares para melhorar atributos.
   * Fundir cartas diferentes para criar cartas de raridade superior.
4. **Visualize os Atributos**: Cada carta mostra os atributos em um gráfico de hexágono, permitindo comparar e planejar suas decisões.
5. **Repita Diariamente**: Novas cartas são disponibilizadas a cada dia, incentivando engajamento contínuo.
